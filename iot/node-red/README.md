# Node-RED Flow

This folder stores Node-RED assets used in AutoGrow.

## Files

- `flows/autogrow-flow.json`: Importable Node-RED flow export.

## How to import

1. Open Node-RED editor.
2. Menu -> `Import`.
3. Select `iot/node-red/flows/autogrow-flow.json`.
4. Deploy.

## Notes

- Remove any real credentials before commit.
- Use environment variables or Node-RED credential store for secrets.
